이 사이트를 선택한 이유

최신 트렌드에 맞는 사이트 디자인을 경험하기 위해 공공사이트를 선택했습니다.
서비스 내용이 한눈에 잘 들어오는 간결한 디자인과 차분한 색감이 마음에 들었습니다.

작업시 중점 사항

html, css 활용 능력 - 섹션 마다 배경색 넣기, table 태그로 표 만들기
jquery 활용 능력 - 탭 기능, click 메뉴(toggleClass) 기능

작업을 마치고

디자인이 복잡하지 않아서 html 구조 만들기가 간편했습니다.
앞으로 제가 사이트 디자인을 한다면 이렇게 해야겠다고 생각했습니다.
